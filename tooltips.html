Tooltips

Tooltips are used to identify or add information to an element. You can use tooltips to show additional information to a button, display alt text of an image, or simply showing a label of an icon.

Placement

Four positioning options are available: top, right, bottom, and left. Besides that, auto option may be used to detect a position that fits the component on the screen.



Usage

Our Tooltips rely on the TooltipModule of ngx-bootstrap library for positioning and other functionality. Tooltips are opt-in for performance reasons, so you must initialize them yourself. See advance usage below.

Manual Trigger

You can manage tooltip using its show(), hide() and toggle() methods. If you want to manage tooltip's state manually, use triggers=""



Dynamic HTML

By using [innerHtml] inside ng-template you can display any dynamic html.
