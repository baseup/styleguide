Form layouts

Forms will by default stack vertically since all form controls applies display: block and width: 100%.

Form groups

The .form-group class is the easiest way to add some structure to forms. Its only purpose is to provide margin-bottom around a label and control pairing. As a bonus, since it’s a class you can use it with <fieldset>, <div>, or nearly any other element.

Inline forms

Use the .form-inline class to display a series of labels, form controls, and buttons on a single horizontal row. Form controls within inline forms vary slightly from their default states.

Custom form controls and selects are also supported.

Horizontal form

Use the grid classes to align labels and groups of form controls in a horizontal layout by adding .form-horizontal to the form (which doesn't have to be a

<form>). Doing so changes .form-groups to behave as grid rows, so no need for .row.



Buttons

Our buttons come with different styles and sizes. We create a button by adding .btn class in a <button> element. However, you can also use this class on <a> or <input> elements (though some browsers may apply a slightly different rendering).

Button sizes

Add .btn-lg, .btn-sm, or .btn-xs for additional sizes.

You can also create block level buttons — those that span the full width of a parent by adding .btn-block.


Floating action buttons

A floating action button that represents the primary action in a page.

Checkboxes and radios

Default checkboxes and radios are improved upon with the help of .checkbox and .radio classes. Checkboxes are for selecting one or several options in a list, while radios are for selecting one option from many.

Disabled checkboxes and radios are supported, you’ll need to add the .disabled class to the parent .checkbox.

Default (stacked)

By default, any number of checkboxes and radios that are immediate sibling will be vertically stacked and appropriately spaced with .checkbox and .radio

Inline

Group checkboxes or radios on the same horizontal row by adding .checkbox-inline and .radio-inline



Static controls

When you need to place plain text next to a form label within a form, use the .form-control-static class on an element of your choice.



Help text

Block-level help text in forms can be created using .help-block.
