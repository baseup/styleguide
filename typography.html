Typography - BaseUp Styleguide

Typography

We follow a 16px base font-size with a 1.5 line-height for our default body text. All other typography font sizes are set on rem unit for easier scaling across device sizes.

Typographic elements do not come with any margins attached to them in order to make them more flexible, but we have utility classes for margin that we can use to add spacing.


Typefaces

Roboto

We use Roboto as our general typeface with font weights ranging from Semibold to Light.


Raleway

We use Raleway mostly in our forms for labels, descriptions and form titles.


Native Font Stack

We also set these Native Font Stack as our fallback typeface:

  • -apple-system for Safari OSX and iOS
  • "Segoe UI" for Windows
  • "Roboto" for Android
  • and "Helvetica Neue", Arial, sans-serif as Basic Web fallback.
