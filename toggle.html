Toggles

Behavior and Style

Use toggles when your intent is to quickly switch between enabled or disabled states (on or off, just like a physical switch). Toggles should never require users to press a button to apply their action. This is because toggle is already a button.

You should only use toggles on settings that need to take effect instantaneously. If a setting requires a button press before it can take effect, you should use a checkbox instead.



Toggle vs Checkbox

A checkbox does not apply a setting at an instant like a switch. Instead, itâ€™s accompanied by a submit button and takes effect after the user presses it.

Checkboxes are useful for applying a batch of settings. With switches, users have to wait for the system to apply a setting one by one. Checkboxes allow the system to handle multiple setting changes at one time.
